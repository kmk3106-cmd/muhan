# 무한매수법 V2.2 - 라오어 원문 vs 코드 비교표

라오어님 원문 설명과 현재 코드 구현을 비교한 표입니다.

---

## 1. T(회차) 계산

| 구분 | 라오어 원문 | 수정 전 코드 | 수정 후 코드 | 비고 |
|------|-------------|--------------|--------------|------|
| **공식** | T = 매수누적액 / 1회매수액 | T = cum_buy_amount / B | 동일 | ✅ 일치 |
| **반올림 규칙** | 소수 둘째자리에서 **올림**하여 소수 첫째자리까지 표기 (반올림 아님) | 소수 둘째자리 **반올림** (ROUND_HALF_UP) | 소수 둘째자리에서 올림→소수 첫째자리까지 표기 | 🔧 수정함 |
| **예시 (0.98)** | 1.0 | 0.98 | 1.0 | ✅ |
| **예시 (1.94)** | 2.0 | 1.94 | 2.0 | ✅ |
| **예시 (2.88)** | 2.9 | 2.88 | 2.9 | ✅ |
| **기준** | 매수 날짜 수 ❌ / 매수누적액 ✅ | cum_buy_amount (체결 합계) | 동일 | ✅ 일치 |

---

## 2. 원금소진 시점 (쿼터모드 진입)

| 구분 | 라오어 원문 | 수정 전 코드 | 수정 후 코드 | 비고 |
|------|-------------|--------------|--------------|------|
| **조건** | T 값이 **39.1 이상 ~ 40 이하** | **39 < T ≤ 40** | **39.1 ≤ T ≤ 40** | 🔧 수정함 |
| **의미** | 1회치를 완전히 매수할 수 없는 상황 | 동일 개념 | 동일 | ✅ |
| **이후** | 쿼터손절 모드 | QUARTER 모드 | 동일 | ✅ |
| **쿼터손절** | 수량의 1/4를 무조건 매도 | MOC 25% (quarter_step≥10 시) | 동일 | ✅ |

**경계값 차이:**
- T=39.05: 원문 기준 쿼터 진입 X / 수정 전 코드 진입 O → 수정으로 원문과 일치
- T=39.1: 원문·코드 모두 쿼터 진입 ✅

## 3. 요약

| 항목 | 수정 여부 | 설명 |
|------|-----------|------|
| T 계산 공식 | ✅ 일치 | cum_buy / B |
| T 올림 규칙 | 🔧 **수정** | 반올림 → 소수 둘째자리에서 올림하여 소수 첫째자리까지 표기 |
| 원금소진 조건 | 🔧 **수정** | 39 < T → 39.1 ≤ T |
| 쿼터손절 1/4 매도 | ✅ 일치 | MOC 25% |
| 전반전/후반전 구간 | ✅ 일치 | T<20 / T≥20 |
| ☆% = 10 - T/2 | ✅ 일치 | A=40 기준 |

## 4. 참고 (2024.09.05 업데이트)

원문에 "(현재는 셋째자리에서 반올림 한 것으로 수정하였습니다. 2024.09.05 update)" 주석이 있으나,  
명시된 예시(0.98→1.0, 1.94→2.0, 2.88→2.9)와 "올림(반올림 아님)" 규칙에 맞춰 **소수 둘째자리에서 올림하여 소수 첫째자리까지 표기**로 구현했습니다.  
추후 원문이 갱신되면 해당 규칙으로 조정 가능합니다.
