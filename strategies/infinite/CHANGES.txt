[2026-02-22] v1.1 - 계좌요약 및 T계산 개선
----------------------------------------
- 예수금(USD) 표시: inquire-present-balance API (CTRP6504R) 연동
  → output2의 frcr_dncl_amt_2 (외화예수금 USD 직접)
- 계좌총액: tot_asst_amt(KRW) / frst_bltn_exrt(환율) = USD 변환
- 환율 표시: 기준환율 + 기준일자 대시보드에 표시
- 모드 표시 개선: NORMAL_전반전, NORMAL_후반전, QUARTER_MOC매도, QUARTER_3/10
- T 계산 수정: T = (매수누적 - 매도누적) / B (매도 시 T 감소)
  → cum_sell_amount 컬럼 추가
- QUARTER 모드 전면 재구성:
  → step 0: MOC 1/4 매도만 (매수 없음)
  → step 1~10: LOC매수 + LOC매도1/4 + LIMIT매도3/4
  → 10회 완료 후: step 0으로 리셋 (MOC 1/4 매도 반복)
  → LOC매도 체결 시: NORMAL(후반전) 복귀
  → quarter_step 증가 로직 추가
  → quarter_base_cash 재계산 (MOC매도 후 cash/10)
- 거래내역: 활성+ON 포트폴리오만 조회/기록
- git 초기화 및 .gitignore 설정
